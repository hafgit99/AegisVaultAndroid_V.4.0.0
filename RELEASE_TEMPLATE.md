# 🛡️ Aegis Vault Android V.4.0.0

Welcome to the most advanced version of **Aegis Vault** yet. This release brings a complete overhaul of the security engine, a stunning new modern UI, and powerful integration features.

## 🚀 Key Highlights

### 🔐 Ironclad Security
*   **Military-Grade Encryption:** Enhanced AES-256-GCM encryption for all sensitive data.
*   **Biometric Access:** Seamless Fingerprint and Face ID integration for instant, secure unlocking.
*   **Argon2 Key Derivation:** State-of-the-art protection against brute-force attacks.

### ✨ New Experience
*   **Glassmorphism UI:** A premium, modern interface designed for focus and aesthetics.
*   **Recycle Bin:** Deleted items are now safely stored in the trash for 30 days.
*   **Localized:** Full support for **Turkish** and **English** with premium translations.

### 🛠️ Functionality
*   **TOTP Authenticator:** Generate 2FA codes directly within the vault.
*   **Cloud Sync:** Encrypted backup and synchronization across devices.
*   **Security Audit:** Automatic password strength analysis and leak detection (HIBP).

## 📦 Installation

1. Download the `app-release.apk` attached below.
2. Enable "Install from Unknown Sources" in your Android settings.
3. Install and set up your secure master password.

> **Note:** For maximum security, we recommend never sharing your master password.
